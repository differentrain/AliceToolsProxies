## Examples

Create a new instance who uses relative path "alice.exe" for invoking.

```csharp
var alicetoolsProxy = new AliceToolsProxy(); 
```

Create a new instance and specifies the path of alice-tools.

```csharp
var alicetoolsProxy = new AliceToolsProxy(@"D:\alice.exe"); 
```

Dump text section of ain files.

```csharp
var ainDump = new AliceToolsAcxDump()
{
    InputPath = "input path.",
    Output = "output path.",
    Option = AliceToolsAinDumpOption.Text
};
AliceToolsOutput output = await alicetoolsProxy.InvokeAsync(ainDump); 
if (output.State != AliceToolsState.Successful)
{
    Console.WriteLine($"Fail: {output.FailReason}");
}
```

Get item list of alice archives.

```csharp
var arList = new AliceToolsArList()
{
    InputPath = "input path.",
};
AliceToolsOutput output = await alicetoolsProxy.InvokeAsync(arList); 
if (output.State != AliceToolsState.Successful)
{
    Console.WriteLine($"Fail: {output.FailReason}");
}
var list = await AliceArchiveItemConverter.Default.ConvertAsync(output.Output)
```
